# homebrew-tap

Homebrew tap for [justEstif](https://github.com/justEstif)'s tools.

## Install

```bash
brew install justEstif/tap/openswarm
```

## Available formulae

| Formula | Description |
|---------|-------------|
| `openswarm` | Multi-agent task orchestration CLI — swarm messaging, tasks, panes, and worktrees |

## Update

```bash
brew update && brew upgrade openswarm
```
